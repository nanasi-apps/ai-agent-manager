# @agent-manager/types

`types` パッケージは、アプリケーション全体で共有される低レベルの型定義を提供するライブラリです。依存関係を最小限に抑えるため、ロジックは含まれていません。

## 主要な型定義

*   **`agent.ts`**: エージェントのステータス定義やインターフェース。
*   **`project.ts`**: プロジェクト（ユーザーがエージェントを実行する対象）に関連する型。
*   **`mcp.ts`**: Model Context Protocol (MCP) に関連する基本的な型定義。
*   **`worktree.ts`**: ファイルシステムやワークツリー操作に関連する型。
*   **`store.ts`**: データストアのスキーマ定義。

## 役割
このパッケージは、循環参照を防ぎ、`shared` パッケージよりもさらに低いレベルで共通の型を定義するために存在します。`shared`, `electron`, `web` すべてのパッケージから参照されます。
